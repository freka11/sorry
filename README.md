# Plead and Promise

A romantic and playful website for asking forgiveness.

## Project info

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
2. **Install dependencies**:
   ```sh
   npm install
   ```
3. **Start the development server**:
   ```sh
   npm run dev
   ```

## Development

The project uses React with Vite for a fast development experience. UI components are built using shadcn-ui and Tailwind CSS.
